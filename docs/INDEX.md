## Scripts Organization

All automation scripts are organized in the `scripts/` directory:

- **`scripts/analysis/`** - Analysis and metrics generation
- **`scripts/bootstrap/`** - Integration and setup scripts
- **`scripts/build/`** - Build optimization tools
- **`scripts/demo/`** - Demonstration and visualization
- **`scripts/entelechy/`** - Entelechy framework tools
- **`scripts/implementation/`** - Implementation automation
- **`scripts/testing/`** - Test execution scripts

See [scripts/README.md](../scripts/README.md) for detailed script documentation.

## Data Organization

Generated data, analysis results, and tracking files are in the `data/` directory:

- **`data/analysis-results/`** - Analysis outputs and metrics
- **`data/cognitive-state/`** - Cognitive system state snapshots
- **`data/entelechy/`** - Entelechy tracking data
- **`data/test-results/`** - Test outputs and logs
- **`data/todo-fixme/`** - TODO/FIXME tracking data

See [data/README.md](../data/README.md) for data management details.
