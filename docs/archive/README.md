# Archive

This directory contains historical documentation, old versions of configuration files, and deprecated materials that are preserved for reference.

## Contents

- **Copilot Memories**: Historical context from early development sessions
- **CMake Variants**: Old versions of CMakeLists.txt for reference
- **Legacy Documentation**: Superseded documentation files

## Purpose

These files are kept for historical context and to understand the evolution of the project, but are not part of the active development documentation.
