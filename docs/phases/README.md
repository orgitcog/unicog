# Phase-Specific Documentation

This directory contains documentation, demos, and coordination files for the 5-phase integration roadmap.

## Integration Phases

1. **Phase I**: Core Extensions (atomspace-rocks, atomspace-restful, moses)
2. **Phase II**: Logic Systems (unify, ure, language-learning)
3. **Phase III**: Cognitive Systems (attention, spacetime)
4. **Phase IV**: Advanced & Learning (pln, miner, asmoses)
5. **Phase V**: Language & Integration (lg-atomese, learn, opencog)

## Contents

- Phase transition coordination scripts
- Phase demonstration code (Scheme, C++, Python)
- Phase-specific implementation details
- Inter-phase bridge documentation
