# Analysis Scripts

This directory contains scripts for analyzing the codebase, tracking metrics, and generating reports.

## Categories

### Placeholder Analysis
- `analyze_placeholders.py` - Analyze TODO/FIXME/STUB markers
- `find_placeholders.py` - Locate placeholder implementations
- `scan_placeholders.py` - Scan for placeholder patterns
- `categorize_placeholders.py` - Categorize placeholders by type

### Code Quality Analysis
- `analyze_cpp_fixmes.py` - Analyze C++ FIXME markers
- `analyze_cpp_fixmes_detailed.py` - Detailed C++ FIXME analysis
- `analyze_test_coverage.py` - Test coverage analysis

### Architecture Analysis
- `analyze_kernel_mapping.py` - Kernel architecture mapping
- `cognitive_membrane_scanner.py` - Cognitive membrane state analysis
- `core_self_awareness_integration.py` - Self-awareness integration analysis
- `dependency_analyzer.py` - Component dependency analysis

### Workflow Analysis
- `analyze_workflows.py` - GitHub Actions workflow analysis
- `analyze_issue_examples.py` - Issue tracking analysis

### Visualization
- `visualize_analysis.py` - Generate analysis visualizations
- `visualize_fixmes.py` - Visualize FIXME distribution
- `generate_analysis_report.py` - Generate comprehensive analysis reports

## Usage

Run these scripts from the repository root to generate analysis reports and metrics for the OpenCog Unified system.
