# Demonstration Scripts

This directory contains scripts for demonstrating OpenCog Unified capabilities and tracking development roadmap progress.

## Scripts

### live_demo.py
Interactive demonstration of cognitive architecture capabilities.

### roadmap_demo.py
Demonstration of roadmap tracking and progress visualization.

### roadmap_tracker.py
Track progress against the development roadmap and generate status reports.

## Purpose

These scripts provide live demonstrations of system capabilities and help visualize development progress for stakeholders and contributors.
